Squirrel Engine

3D C++ Game Engine

  • Solo
  • C++
  • 3D
  • DirectX11
  • Scene Graph
  • Platform Independency
  • Custom Memory Allocators
  • Physics Integration
  • Serialization
  • Documentation

Squirrel Engine is a C++ 3D game engine. It is currently a work in progress solo project which so far uses scene graph representation with a node like object structure to create games. Similar to that of Godot. It makes use of Direct X 11 for rendering which is abstracted away for platform independence. Currently the engine supports scene serialization, physics, threaded loading, custom memory management for performance, and basic lighting.
