2nd Year Graphics Project

OpenGL Fixed Function Pipeline Project

  • C++
  • OpenGL
  • Git
  • 3D
  • Asset Loading
  • Lighting
  • Mesh Generation
  • Project Management

At the end of the first semester in second year we were tasked with creating an application which showcased our OpenGL graphics knowledge. I decided to challenge myself with this project working on a series of 5 rooms which showcased different 3D graphics principles I had learned over the semester. My most notable achievement from this project would have to be my shadow volumes using Carmack's reverse and configuring them as such to work in reflections too. I also showcased other graphics principles like depth based alpha blending, billboarding, lighting techniques, mesh loading, mesh generation and UV work. I have learned a lot about the fundamentals of graphics programming from this project and it has helped me in future projects with a better understanding of how graphics systems work in modern game engines.

This Project Features:

  • Procedural model generation
  • Custom Shadow Volumes
  • Stencil Reflections
  • Depth based alpha blending
  • Static and dynamic shadows
  • Billboarding
  • Mesh loading from file
  • Custom Mesh, Material & Light classes
  • Skybox techniques
