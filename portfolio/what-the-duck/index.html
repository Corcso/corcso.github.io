What The Duck!?

Global Game Jam 2024

  • C#
  • Unity
  • Git
  • Local Multiplayer
  • 2D

What the Duck!? Is a button masher game inspired by the childhood game Duck Duck Goose which had players pressing changing keys to outrun each other. The game was made for the game jam's theme of "Make Me Laugh". I worked in a large team of 7, 3 Artists, 3 Programmers, and 1 Designer. I was tasked with the source control management for this project alongside development. I worked on various systems for the game, notably the game manager and the chase scene. Working on this project I learned about effective ways to structure a game, particularly in unity, as well as skills relating to local multiplayer.

This Game Features:

  • Local multiplayer
  • Character selection
  • 3-6 Players
  • Multiple sharacter sprites
  • Full menu and how to play section

The game's executable, source code and github repository can all be found on its Global Game Jam page here!


Many thanks to Jaihui He, Karina, Thomas McGregor, Noah Morrison, Jerry Fraser and Thomas Richards for their work on the game as part of our Global Game Jam 2024 team.
