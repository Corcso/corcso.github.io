Karate Kevin

AGDS Freshers Jam 2022

  • C#
  • Unity
  • Git
  • 2.5D

Karate Kevin is a game where you play as a seagull who attacks members of the elderly in a karate fashion for sausage rolls. This was my first ever game jam where I was taken under the wing of a team of four fourth year students. The experience was invaluable, I learned lots about game development particularly in unity which helped accelerate my coursework in first year. I was tasked with the player movement and attacking system as well as various other pieces of UI and polish. This project helped me learn a lot about game development fundamentals as well as source control.

This Game Features:

  • Combo attacks
  • Progressive difficulty
  • Infinite map
  • Controler/Gamepad Support
  • Online leaderboard
  • NPC Attacking AI

The game's executable can be found on the itch.io page here!
